Supported Features:
  • Linear and time-based X-axis
  • Linear and logarithmic Y-axis
  • Multiple data series
  • Customizable colors and styling
  • Grid lines and labels
  • Points and fill areas
Tips:
  • Use transparent background for web embedding
  • Adjust margins to fit axis labels
  • Set point radius to 0 to hide data points
  • Use fill alpha for area charts
