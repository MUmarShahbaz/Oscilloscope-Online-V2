How to Use
Serial Connection:
  1. Click "Connect" to select your serial port
  2. Choose the appropriate baud rate
  3. Click "Start" to begin plotting
Data Format:

Send data in the format: value1/value2/value3/value4

Each value must be separated by the configured break character

Each line should end with a newline character (\n)
